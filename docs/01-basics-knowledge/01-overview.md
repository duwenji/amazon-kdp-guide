## 🎯 Amazon KDP（キンドル・ダイレクト・パブリッシング）とは

### **定義**

**Amazon KDP** = Amazon が提供する **電子書籍・紙の本の自主出版プラットフォーム**

```
あなたの原稿
  ↓
KDP にアップロード
  ↓
Amazon Kindle ストアで販売
  ↓
読者が購入 → あなたが報酬獲得
```

### **簡単に言うと**

「自分で書いた本を、Amazon を通じて世界中に販売できるサービス」

---

## 💡 KDP でできる 5 つのこと

### 1️⃣ **Kindle 電子書籍の出版**

**最も一般的な使い方**

```
EPUB / PDF / Word 形式の原稿
  ↓
KDP にアップロード
  ↓
Kindle ストア で販売開始
    - Kindle Paperwhite
    - Kindle Oasis
    - iPhone / iPad の Kindle アプリ
    - Windows / Mac の Kindle アプリ
    - ブラウザの Kindle Cloud Reader
```

**対応フォーマット:**
- EPUB（推奨）
- PDF（固定レイアウト向け）
- Microsoft Word（.doc / .docx）
- HTML

## 🗂️ 電子書籍フォーマットの基礎（EPUB / PDF / KDP 変換）

### EPUB
- 国際標準（W3C）が定める電子書籍フォーマット
- 再フロー型で、多種多様な端末に対応
- KDP では推奨フォーマットで、アップロード時に最適化処理がされる
- 特徴: CSS によるスタイル調整、目次（ナビゲーション）

### PDF
- 固定レイアウト確認や印刷レビューに向く
- 図版や紙面イメージを保ちたい技術資料で有効
- リフロー型電子書籍の本文入稿は `EPUB` を優先

### KDP 側の内部変換
- KDP に `EPUB` をアップロードすると、Amazon 側で Kindle 向け内部形式へ最適化される
- そのため、原稿作成段階では `EPUB` と確認用 `PDF` を用意しておけば十分
- 旧来の Kindle 専用形式は、新規ワークフローでは通常不要

> 体系的な流れ
> 1. 執筆は Word/Markdown/HTML などで行う
> 2. `EPUB` を生成して本文入稿用に使う
> 3. `PDF` を生成して固定レイアウトや印刷見えを確認する
> 4. `*-kdp-registration.md` を見ながら KDP にメタデータを転記する

### 2️⃣ **Kindle Unlimited（読放題プログラム）に参加**

**追加収入を得るしくみ**

```
Kindle Unlimited（読放題）に登録
  ↓
読者が購読料（日本: 980 円/月）で利用
  ↓
あなたの書籍が読まれた時間に応じて報酬
  ↓
月間プール（約 $40 million）から配分
```

**メリット:**
- ✅ 購入者以外からも収入
- ✅ 読放題ユーザーへのリーチ
- ✅ 購入 + 読放題 = ダブル収入

**デメリット:**
- ❌ 独占契約（他のプラットフォーム販売禁止）

---

### 3️⃣ **紙の本を出版（POD）**

**同じプラットフォームで紙の本も販売**

```
Kindle 電子書籍 + 紙の本を同時出版
  ↓
読者が紙の本を注文
  ↓
Amazon が自動で印刷・配送
  ↓
あなたが利益を獲得
```

**メリット:**
- ✅ 信頼度アップ（「紙の本も出している」）
- ✅ 図書館への納入申請が可能
- ✅ 販売チャネルの拡大

**コスト:**
- 無料（Amazon が自動印刷）
- 利益は「定価 - Amazon の印刷費」

---

### 4️⃣ **複数冊の書籍を管理**

**1 つのダッシュボードで複数の本を管理**

```
KDP ダッシュボード
  ├── 書籍①: クリーンアーキテクチャ
  ├── 書籍②: SOLID 原則ガイド
  ├── 書籍③: デザインパターン解説
  └── 書籍④: テスト駆動開発
```

**機能:**
- ✅ 各書籍の売上・ダウンロード数を個別追跡
- ✅ プロモーション / セール実施
- ✅ 一括で複数冊を無料化

---

### 5️⃣ **全世界で販売**

**170+ の国と地域で自動的に販売**

```
日本の読者
  ↓
米国（英訳版など）
  ↓
ヨーロッパ
  ↓
アジア太平洋地域
  ↓
すべて自動で配信・販売
```

**メリット:**
- ✅ 国際的な著者になれる
- ✅ 多言語出版も可能
- ✅ 複数の通貨で販売

## 📊 Amazon KDP の 3 つの収益モデル

### **モデル A：通常販売（Kindle ストア）**

**最もシンプル・最初はこれ推奨**

```
あなたが価格を設定（¥99～¥99,999）
  ↓
読者が購入
  ↓
Amazon が 30～35% 取得
  ↓
あなたが 65～70% を獲得

例：
¥1,000 で販売
  → あなたの収入 = ¥700
```

**印税率の仕組み:**
- 定価 $2.99～$9.99（約¥300～¥1,000）→ **70% 獲得**（最高）
- 定価 $0.99～$2.98 または $10.00 以上 → **35% 獲得**

**特徴:**
- ✅ 最高の印税率（他の出版社より高い）
- ✅ 価格を自由に設定
- ✅ 他のプラットフォームとの併売可

---

### **モデル B：Kindle Unlimited（読放題）**

**追加収入・読者層の拡大**

```
書籍を「Kindle Unlimited」に登録
  ↓
読者は月 980 円で読み放題購読（日本）
  ↓
あなたの書籍が読まれたページ数に応じて報酬
  ↓
月間プール（約 $40 million）から配分
```

**報酬計算例:**
```
月間プール: $40,000,000
全書籍の読まれたページ数: 10 億ページ

あなたの書籍: 100,000 ページが読まれた
  ↓
あなたの報酬 = $40,000,000 × (100,000 / 1,000,000,000)
           = $4,000（約 ¥440,000）
```

**条件:**
- ⚠️ **独占契約**: Kindle Unlimited に登録した本は、他のプラットフォーム（Apple Books, Google Play など）での販売禁止

**向いている書籍:**
- ✅ ストーリー性がある（読み続けられやすい）
- ✅ ページ数が多い（読まれるページ数が増える）
- ✅ 頻繁に読み直される（再利用価値がある）

---

### **モデル C：POD（紙の本販売）**

**信頼度アップ・多様な販売チャネル**

```
Kindle 版 + Print On Demand（紙版）を販売
  ↓
読者が紙の本を注文
  ↓
Amazon が自動で印刷・配送
  ↓
定価 - 製造原価 = あなたの利益
```

**コスト:**
- 例：定価 ¥2,000、製造原価 ¥800
- あなたの利益 = ¥1,200 / 冊

**メリット:**
- ✅ 「著者」としての信頼度 UP
- ✅ ビジネス資料としての価値 UP
- ✅ 図書館への寄贈・申請が可能

**デメリット:**
- ❌ 印刷コストがかかる（売れなければ赤字）
- ❌ 配送に 3 日～1 週間かかる

## ⚖️ Amazon KDP vs その他プラットフォーム

### **主要プラットフォーム比較**

| 項目 | Amazon KDP | Apple Books | Google Play | 出版社経由 |
|------|-----------|-----------|-----------|----------|
| **登録料** | 無料 | 無料 | 無料 | 数万円～ |
| **ISBN** | 自動割り当て | 必須購入 | 不要 | 提供 |
| **印税率** | 70%（最高） | 70% | 70% | 10～15% |
| **販売地域** | 全世界 | 全世界 | 全世界 | 限定 |
| **出版速度** | 24 時間以内 | 1～2 週間 | 1～2 週間 | 数ヶ月 |
| **Kindle Unlimited** | ✅ あり | ❌ なし | ❌ なし | ❌ なし |
| **紙の本** | ✅ POD 可 | ✅ POD 可 | ❌ 不可 | ✅ 可能 |
| **複数プラットフォーム** | ❌ 独占禁止* | ✅ 可能 | ✅ 可能 | ✅ 可能 |

**\* Kindle Unlimited に参加した場合のみ独占契約**

## ✅ Amazon KDP の メリット（9 つ）

### 1. **完全無料**
```
- 登録：¥0
- ファイルアップロード：¥0
- 販売：¥0
- 全て無料スタート
```

### 2. **高い印税率**
```
- 70% のロイヤリティ（業界最高水準）
- 出版社経由（10～15%）より圧倒的に高い
- あなたが大部分の売上を獲得
```

### 3. **即座に販売開始**
```
- 審査なし（内容がアマゾンの規約を守っていれば OK）
- 数時間～24 時間で Amazon Kindle ストアに掲載
- 従来の出版：数ヶ月～1 年かかる
```

### 4. **世界規模で販売**
```
- 170+ の国と地域で販売
- 言語ごとに異なる価格を設定可
- 国を跨いだ販売収入
```

### 5. **ISBN が無料**
```
- Amazon が自動で ISBN を割り当て
- ISBN 単独購入：¥1000～¥2000
- KDP: 完全無料
```

### 6. **Kindle Unlimited で追加収入**
```
- 購入者とは別の読放題ユーザーから収入
- 月間プール（$40M+）から配分
- 読まれるほど収入増加
```

### 7. **いつでも修正・更新**
```
- ファイルを差し替え可能
- 説明文をいつでも変更可
- 価格をいつでも変更可
- 修正から反映まで数時間
```

### 8. **詳細な売上分析**
```
- 日単位の売上追跡可
- 地域別・価格帯別の分析
- Kindle Unlimited ページ数の追跡
```

### 9. **マルチフォーマット対応**
```
- Kindle（電子）
- 紙の本（POD）
- 音声本（Audible）*
- 複数形式で同時販売
```

---

## ❌ Amazon KDP の デメリット（5 つ）

### 1. **Kindle Unlimited は独占契約**
```
- Kindle Unlimited に登録すると独占契約に同意
- Apple Books, Google Play では販売不可
- 他のプラットフォームの自由度がなくなる
```

### 2. **Amazon 専用の ISBN**
```
- KDP の ISBN は Amazon 専用
- 他のプラットフォームでは使用不可
- 複数プラットフォーム販売には別途 ISBN 購入が必要
```

### 3. **競争が激しい**
```
- 毎日数千の書籍がアップロードされる
- タイトル決定が重要（検索に引っかかるか）
- マーケティングが自分で必要
```

### 4. **マーケティングは自分で**
```
- Amazon は宣伝してくれない
- SNS / ブログ / メールでの自主宣伝必須
- 初期のレビュー獲得が困難
```

### 5. **Amazon の方針変更**
```
- Amazon の利用規約が変わる可能性
- 価格設定ルールの変更
- プラットフォーム政策の急変
```

## 📝 KDP に向かない場合

### **こういう人は別の選択肢を検討**

- 😞 費用をかけてでも丁寧な編集・デザインが欲しい
  → **従来出版社経由** の出版
  
- 😞 複数のプラットフォーム（Apple Books, Google Play）で同時販売したい
  → **独立出版社**（IngramSpark など）を検討、または部分的に ISBN 購入
  
- 😞 Amazon 以外（例：日本のみ販売）に特化したい
  → **ローカル出版社**と契約
  
- 😞 マーケティングが嫌 / 得意でない
  → **従来出版社**に全てお任せ

## 📞 よくある質問（このセクション終了時点）

**Q：本当に 70% ロイヤリティもらえる？**

A：はい。ただし条件があります：
- 定価が ¥299～¥9,999 の場合のみ 70%
- それ以外（¥99～¥298 または¥10,000 以上）は 35%
- 初期設定で 70% を選択する方法は [03-advanced-topics/01-metadata-and-pricing.md](../03-advanced-topics/01-metadata-and-pricing.md) で詳しく説明

---

**Q：Kindle Unlimited と通常販売、どっちにすべき？**

A：迷っているなら、まず **非参加（通常販売）で始める** ことをお勧めします。理由：
- 他のプラットフォームでも販売可能
- いつでも Unlimited に切り替え可能（90 日契約ごと）
- データを見てから判断できる

詳しくは [03-advanced-topics/02-kindle-unlimited.md](../03-advanced-topics/02-kindle-unlimited.md) を参照。

---

**Q：green-architecture 書籍の場合、通常販売？ Unlimited？**

A：推奨は **通常販売**（Kindle Unlimited 非参加）です。理由：
- 技術情報は時間とともに陳腐化
- 頻繁な更新が必要（Unlimited だと修正困難）
- Apple Books など複数プラットフォーム展開を視野に
