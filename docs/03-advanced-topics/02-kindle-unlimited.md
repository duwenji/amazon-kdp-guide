## 🎯 Kindle Unlimited とは

### **定義**

```
Kindle Unlimited （KU）= Amazon の「読放題」プログラム

読者側:
  月額 ¥980 で、200 万冊以上が読み放題

著者・出版社側:
  読者が読んだページ数に応じて報酬獲得
```

### **KU の全体像**

```
読者（購読者）
  ↓ 月額 ¥980 を支払い
  ↓
Amazon Kindle Unlimited プール
  ↓ 月間約 $40 million（約 600 万円）を配分
  ↓
著者・出版社（あなた）
  ↓ ページ読破数に応じて分配
  ↓
あなたのアカウント（日本）
  ↓
月末に振込（例: 月 5 万円など）
```

## 💡 Kindle Unlimited に参加するメリット・デメリット

### **メリット✅**

#### 1️⃣ **新しい収入源**

```
通常の販売：
  読者が ¥1,490 で購入
  → あなたのロイヤルティ: ¥1,043 (70%)

KU での読破:
  読者は月額 ¥980 利用
  → 月間プール $40M を全著者で分配
  → あなたが 100 ページ読まれた場合: 約 ¥100～300
```

#### 2️⃣ **読者リーチの拡大**

```
通常販売：
  購入をためらう層 → 購入しない

KU 参加：
  購入をためらう層 → 読放題で試し読み
  → 1000 ページ読まれたら: 約 ¥1,000～3,000
```

#### 3️⃣ **Kindle Unlimited 専用マーケティング機能**

```
Amazon が KU に未登録の本より KU 本を優先表示
  → 検索順位が高くなりやすい ✅
  → クリック数（ページビュー）が増える
  → その結果、販売数が増える
```

#### 4️⃣ **借入（バロー）に対応**

```
KU では「購入」ではなく「借入」

読者が「借りる」感覚 = 購入ハードル低い
  → 購入数が増える可能性
```

### **デメリット❌**

#### 1️⃣ **独占契約の強制**

```
KU に登録すると:
  その本を【KU 専用】にする必要があり
  ↓
  他のプラットフォームでは販売禁止：
    ❌ Apple Books（iBooks）
    ❌ Google Play Books
    ❌ Kobo
    ❌ 自社ウェブストア
    
【重要】日本以外（米国など）での販売も不可
  → 世界中の Kindle.com でのみ販売可能
```

#### 2️⃣ **報酬が不確定**

```
販売での報酬（確定的）：
  $9.99 で販売 → ロイヤルティ $6.99（70%）
  【確実】

KU での報酬（不確定）：
  月間プール $40M / 全著者が読んだページ数
  → 月ごとに変動（平均 $0.004～0.005/ページ）
  → 月によって大きく変わる可能性
    （プール額が減ると報酬も減少）
```

#### 3️⃣ **短期的には売上が減る可能性**

```
シナリオ：

【参加前】
  月 50 冊 × ¥1,043 = ¥52,150

【参加直後】
  月 200 回借用 × 500 ページ読破
  = 月間 100,000 ページ
  = ¥40,000～500,000（変動大）

参加直後は販売数が落ちることが多い
  → 読者が「購入」から「借入」に切り替え
  （ただし、上手くいけば総売上が増える）
```

## 📋 KU 参加の判断フロー

### **参加すべきか迷った場合の判断表**

| 状況 | 判断 | 理由 |
|------|------|------|
| **初出版で知名度ゼロ** | **KU 推奨** ✅ | リーチ拡大で販売数増加 |
| **既に複数冊ベストセラーがある** | **非参加推奨** | 独占契約が足かせ |
| **複数言語（日本語＋英語）での販売戦略** | **非参加推奨** | 海外での販売が制限 |
| **シリーズ化予定（第 2 巻、第 3 巻）** | **KU 推奨** | シリーズ全冊を KU で |
| **技術書（アップデート多い）** | **非参加推奨** | 更新版を別の販路で売りたい |

## 🚀 ステップ 1: Kindle Unlimited への参加申請

### **申請手順**

#### **1. KDP ダッシュボールを開く**

```
https://kdp.amazon.com → 著者用ダッシュボール
```

#### **2. 既存本を編集モードで開く**

```
「個別に本を管理」
  ↓
登録済みの本（例: clean-architecture.azw3）をクリック
  ↓
「編集」ボタンをクリック
```

#### **3. KU セクションで「登録」をクリック**

```
【本の内容】セクション
  ↓
「Kindle Unlimited に登録」
  ↓
「利用規約に同意して登録」をクリック
```

#### **4. KU 独占契約に同意**

Amazon が表示するポップアップ：

```
「この本を 90 日間 KU 専用にします。
その間、他のプラットフォームでは販売できません。
同意しますか？」

→ 「はい」をクリック
```

### **申請後の流れ**

```
申請直後（数分以内）:
  「Kindle Unlimited に登録されました」と表示

確認:
  ≈ 数時間～1 日後
  → ダッシュボールの「登録ステータス」が更新
  → 「Kindle Unlimited に登録中」

本格開始:
  ≈ 登録から 24～48 時間後
  → Kindle Unlimited の検索対象に追加
  →読者が「KU で読む」ボタンで読み始める
```

## 📊 ステップ 2: KU での報酬計算

### **KU での報酬の仕組み**

```
【月間プール】:
  毎月 Amazon が $40 million を割り当て
  （例: 2024 年 3 月 = $41.5M）

【配分方法】:
  プール総額 ÷ 月間全ページ読破数 = 1 ページあたりの報酬

例: 2024 年 3 月（実例）
  プール: $41,500,000
  全著者の読破ページ数: 4 billion ページ
  1 ページあたり: $0.0104
  
  あなたの本が 10,000 ページ読まれた:
  報酬 = 10,000 × $0.0104 = $104

【重要】:
  プールが減ると報酬も減少
  → 毎月変動する（当てにならない側面あり）
```

### **報酬の実例**

#### **シナリオ: クリーンアーキテクチャ本の例**

```
書籍: クリーンアーキテクチャ完全ガイド
ページ数: 500 ページ

月 1: 100 人が 100% 読破
  = 100 × 500 = 50,000 ページ
  = 50,000 × $0.0104 = $520 ≈ ¥78,000

月 2: 150 人が 80% 読破
  = 150 × 400 = 60,000 ページ
  = 60,000 × $0.0105 = $630 ≈ ¥94,500

月 3: 50 人が 100% 読破
  = 50 × 500 = 25,000 ページ
  = 25,000 × $0.0098 = $245 ≈ ¥36,750
  （プール額が減った月）
```

### **KU 報酬の確認方法**

```
KDP ダッシュボール → 「レポート」
  ↓
「KU レポート」をクリック
  ↓
以下が表示:
  ・ページビュー数（日別）
  ・読破ページ数（日別）
  ・推定報酬（日別）
  ・月間合計
```

## 🎯 KU vs 単独販売の比較

### **収入モデルの比較**

#### **シナリオ: 月 100 冊売れる本の場合**

**【単独販売】（KU 非参加）**

```
販売数: 100 冊/月
定価: ¥1,490
ロイヤルティ: 70% = ¥1,043/冊

月間売上: 100 × ¥1,043 = ¥104,300

特徴:
✅ 収入が確定的
✅ 他販路での販売可能
❌ KU 読者にリーチできない
```

**【KU 参加】（参加戦略）**

```
販売数: 30 冊/月
KU・読破: 10,000 ページ/月

販売売上: 30 × ¥1,043 = ¥31,290
KU 報酬: 10,000 × ¥9.5 = ¥95,000
（1 ページ平均 ¥0.0095 と仮定）

月間売上: ¥31,290 + ¥95,000 = ¥126,290

特徴:
✅ 総売上が増加（単独販売比 121%）
✅ KU 専用マーケティング機能の恩恵
❌ 独占契約で他販路にバラれない
❌ 報酬が月ごとに変動
```

### **どちらを選ぶべきか?**

```
✅ KU 参加すべき場合:
  • 初出版で知名度がない
  • 販売数 50 冊/月 以下
  • 日本マーケット中心
  • シリーズ化予定

❌ 単独販売すべき場合:
  • 既に知名度がある著者
  • 月間販売数 100 冊以上
  • 国際販売を重視
  • 技術書で頻繁に更新
```

## 📈 ステップ 3: KU での成功戦略

### **KU に適した書籍の特性**

```
KU で成功しやすい書籍:

✅ シリーズもの
  例: 「クリーンアーキテクチャ」「SOLID 原則」「デザインパターン」
  理由: 第 1 巻を KU で読破 → 第 2 巻も読んでくれる

✅ ニッチジャンル（競争相手が少ない）
  例: 「Rust で学ぶ言語処理」
  理由: 検索上位を狙いやすい

✅ エンゲージメント高い本
  例: 実装ガイド、チュートリアル本
  理由: 最後まで読破される確率が高い

❌ KU に不向きな書籍:

❌ リファレンス本（最後まで読まない）
  例: API リファレンス
  理由: ページ読破率が低い

❌ 既に売上が安定している本
  例: ベストセラー本
  理由: 独占契約がデメリット

❌ 国際市場向け
  例: 英語版を予定している
  理由: 独占契約で国際販売が制限される
```

### **KU での販売・読破を増やす施策**

#### **1️⃣ シリーズ化戦略**

```
書籍エコシステム構築:

第 1 巻: 「クリーンアーキテクチャ入門」
  ↓ KU で低価格提供
  ↓ 読者が完全読破
  ↓

第 2 巻: 「SOLID 原則実装ガイド」
  ↓ KU で提供（まとめ売りで KU トリプル報酬狙い）
  ↓ 既読者が購買率 80%
  ↓

結果: シリーズ全体で月 200,000 ページ読破
```

#### **2️⃣ ページ計算上の最適化**

```
KU では「ページ数」が報酬に直結:

ページサイズの選択:
┌─────────────────────────────┐
│ Word 版: 1 ページ = 250 語   │→ 少ない報酬
│ EPUB 版: 1 ページ = 260 語   │→ 標準
│ PDF 版: 1 ページ = 300 語    │→ 少ない報酬
│ HTML 版: 1 ページ = 変動大   │
└─────────────────────────────┘

戦略:
✅ AZW3 / EPUB での提出（ページ数が多くカウント）
❌ PDF は避ける（ページ数が圧縮される）
```

#### **3️⃣ 序文・導入部分を充実させる**

```
KU では最初の数ページが勝負:

読者が最初の 10% で放棄
  → 報酬ゼロ


読者が最後まで読破
  → 満額報酬（例: 500 ページ分）

対策:
✅ 最初の章で引き込む
✅ 目次で期待値を高める
✅ 導入部分を充実させる
```

#### **4️⃣ マーケティング連携**

```
KU タイトルをプロモーション:

SNS: 「KU で読放題「クリーンアーキテクチャ - 読み放題」
ブログ: 「これもう無料で読んでいいの？」
メール: 購読者リストに通知

結果: 初週の読破数が 5 倍に増えることも
```

## ⏰ ステップ 4: 90 日契約と更新

### **KU 契約期間**

```
初回購読: 90 日間（3 ヶ月）
その後: 30 日ごとの自動更新
```

### **契約更新の管理**

#### **更新をスキップしたい場合**

```
方法:
1. KDP ダッシュボール開く
2. 該当本を編集モード
3. 「KU 登録を解除」をクリック
4. 90 日間の契約終了時点で自動解除

注意:
・ 解除された時点で AWS オンライン店で販売可能になる
・ ただし 90 日間はそれ以前に販売を始められない
・ 他販路（Apple Books など）での販売も 90 日後から
```

#### **更新を継続する場合**

```
何もしない → 自動で 30 日ごとに更新

メリット:
✅ 自動でメンテナンス不要
```

## ❓ KU よくある質問

### **Q1: KU に登録しても販売数は減らない？**

A: 基本的には減ります。理由：
```
読者が「購入」から「借入」に切り替え
  → 購入冊数が 30～70% 減少
  → ただし、KU による報酬がカバーできれば OK
```

### **Q2: 一度 KU に登録したら、やめられない？**

A: やめられます。ただし条件があります：
```
90 日の契約終了時点で「更新をスキップ」
  → 自動的に KU から除外
  → 他販路での販売が可能に

ただし一度 KU をやめると、再登録は 90 日後以降
```

### **Q3: KU に登録した本を他の言語（英語版など）で出版できる？**

A: はい、別タイトルなら OK：
```
日本語版: 「クリーンアーキテクチャ完全ガイド」(KU 登録)
英語版: 「Clean Architecture Complete Guide」(別タイトル、KU なし)
```

### **Q4: KU でも価格設定できる？**

A はい。ただしルールあり：
```
価格: $2.99～$9.99 の範囲で設定可能
注意: KU では販売数が減るため、価格を下げるのが定石
```
