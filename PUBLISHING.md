# PUBLISHING

## Pre-publish flow
1. `./.github/skills-config/ebook-build/invoke-build.ps1` を実行する
2. 以下の成果物を確認する
   - `ebook-output/amazon-kdp-guide.epub`
   - `ebook-output/amazon-kdp-guide.pdf`
   - `ebook-output/amazon-kdp-guide-kdp-registration.md`
3. `README.md` / `docs/index.md` / `MASTER-INDEX.md` の主要リンクを確認する
4. KDP 手順・価格情報・カテゴリ・説明文が `*-kdp-registration.md` と一致しているか確認する
5. GitHub Pages 表示を確認して公開する
