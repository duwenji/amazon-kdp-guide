## ステップ 1️⃣: Amazon アカウントを確認 / 作成

### **既に Amazon アカウントがある場合**

```
👉 スキップ → ステップ 2 へ
（既存のメールアドレス・パスワードで KDP にサインイン）
```

### **Amazon アカウントがない場合**

**URL**: https://account.amazon.com/

```
1. 「新規登録」をクリック
   
2. 以下を入力
   ✓ 名前（日本語でも OK）
   ✓ メールアドレス（GmailやYahoo メールなど）
   ✓ パスワード（8 文字以上、英数字・記号を含む）
   
3. 「アカウントを作成」をクリック
   
4. メール認証
   ✓ 入力したメールに Amazon からメールが届く
   ✓ メール内のリンクをクリック
   ✓ 確認完了

✅ Amazon アカウント完成！
```

### **パスワード設定のコツ**

```
❌ 避けるべき：
  - 誕生日のみ
  - 単純な数字（123456）
  - 辞書に載っている単語だけ

✅ 推奨：
  - Aj92!kl@m（英字大小・数字・記号混在）
  - Pnx8@qwE9v
  
💡 ツール：パスワード管理アプリ
  - 1Password / LastPass / Bitwarden（無料版あり）
  - 複数のパスワードを安全に管理
```

## ステップ 2️⃣: Kindle Direct Publishing にサインアップ

### **KDP にアクセス**

**URL**: https://kdp.amazon.com/

```
1. 左上の「ログイン」をクリック
   
2. Amazon アカウントでサインイン
   ✓ メールアドレス
   ✓ パスワード
   
3. 利用規約を確認
   ✓ 「KDP にようこそ」ページが表示
   ✓ 利用規約を読む（重要）
   ✓ 「同意する」をクリック
   
4. 出版者情報ページ へ → ステップ 3 へ
```

## ステップ 3️⃣: 出版者プロフィール を設定

### **出版者名（著者名）を決定**

**これは本の表紙に表示される名前です！重要。**

```
KDP ダッシュボード
  → 「アカウント」 タブをクリック
  → 「著者プロフィール」セクション
```

#### **著者名の種類**

**① 本名を使う（推奨：最初はこれ）**
```
メリット:
  - 信頼度が高い
  - SNS と連動しやすい
  
例:
  杜 文吉
```

**② ペンネームを使う**
```
メリット:
  - プライバシー保護
  - ブランド作成
  - 複数ジャンルの分離（推理小説 vs 技術書）
  
例:
  山田太郎
  (実名: 田中花子)
```

#### **著者名キーポイント**

- ✅ 短く覚えやすく（10 文字以下推奨）
- ✅ ユニークな名前（他と被らない）
- ✅ スペルが簡単（読者が検索しやすく）
- ✅ 複数冊出版予定なら統一

**例：良い著者名**
```
✅ 清水勇太
✅ Dr. Yamamoto
✅ システム太郎
❌ aaaaaaaaaaaa（意味不明）
❌ 著者名は後で変更可能（初回登録時）
```

### **著者プロフィール を入力**

```
①著者名入力
  入力欄：「著者名」
  入力内容：日本語でも英数字でも OK
  
  例：杜 文吉
  または
  例：Bunkyichi To
  
②自己紹介文を入力（オプション）
  入力欄：「著者について」
  文字数：最大 500 文字
  
  📌 例：
  「ソフトウェアアーキテクチャの専門家です。
  クリーンアーキテクチャの実装方法や
  SOLID 原則について 15 年の実務経験があります。」
  
③プロフィール画像をアップロード（オプション）
  推奨サイズ：500 x 500 ピクセル
  ファイル形式：JPG / PNG
  
  💡 ツール：
  - Canva（https://www.canva.com）
  - 簡単におしゃれな画像を作成可

④ ソーシャルメディアリンク（オプション）
  Twitter / Instagram / ウェブサイト URL
  追加することで、読者があなたをフォローしやすくなる
  
  例：
  - Twitter: @yourhandle
  - ウェブサイト: https://yoursite.com
```

### **保存方法**

```
全て入力したら
  ↓
「プロフィールを保存」 をクリック
  ↓
✅ 「プロフィールが更新されました」
  と表示されれば完了
```

## ステップ 4️⃣: 支払い情報（ロイヤリティアカウント）を登録

**最重要ステップ：ここを設定しないと報酬が受け取れません！**

### **設定方法**

```
KDP ダッシュボード
  → 「アカウント」 タブ
  → 「ロイヤリティアカウント」セクション
```

### **入力する情報**

#### **1. 銀行口座情報**

```
入力項目：
  ✓ 銀行名（例：三菱 UFJ 銀行）
  ✓ 支店名（例：渋谷支店）
  ✓ 口座種別（例：普通）
  ✓ 口座番号（7～8 桁）
  ✓ 口座名義（銀行に登録された通り）
  
💡 ポイント：
  - 口座名義は本人確認書類と同じ名前に
  - 複数の銀行を登録可能（複数冊出版の場合）
  - 1 冊目＝確認用に時間がかかる（1～2 週間）
```

#### **2. 住所・電話番号**

```
入力内容：
  ✓ 名前（本名）
  ✓ 郵便番号
  ✓ 住所（都道府県・市区町村・番地）
  ✓ 電話番号（09x-xxxx-xxxx）
  
💡 ポイント：
  - 住所は銀行登録情報と一致させる
  - 電話番号は携帯 / 固定電話, どちらでも OK
```

#### **3. 税情報**

```
質問: 「あなたは US の税法下での居住者ですか?」

日本在住ならば
  → 「いいえ」を選択
  
その後：
  → ITIN（個人納税者識別番号）の入力が求められるかも
  → 日本在住なら通常不要
  → 「この時点では提供できません」で OK
```

### **支払い方法の確認**

```
報酬の受け取り方法：
  
💴 日本の場合：
  1. 銀行振込（推奨）
     - KDP が月 1 回, あなたの銀行口座に振込
     - 着金まで 3～5 営業日
     - 手数料なし（Amazon 側）
     
  2. Amazon アカウントクレジット（オプション）
     - Amazon.co.jp での買い物に使える
     - 銀行振込より早い
```

### **保存と確認**

```
全て入力したら
  ↓
「保存」ボタンをクリック
  ↓
確認メール受信
  → メール内のリンクをクリック（認証）
  ↓
✅ ロイヤリティアカウント有効化
```

## ステップ 5️⃣: 税務情報（W-8BEN）の登録（時間がかかる場合）

### **概要**

```
KDP（Amazon 米国）から報酬を受け取る際,
米国税務フォーム W-8BEN を提出する必要がある場合があります。
（日本在住の場合のみ）
```

### **W-8BEN とは**

```
= 「私は米国居住者ではない」という宣言フォーム
= 米国の税法下では適用されません

日本在住なら
  → US Tax ID（ITIN）は不要
  → 提出は簡単
```

### **登録方法**

```
KDP ダッシュボード
  → 「アカウント」タブ
  → 「税務情報」セクション
  
↓

フォーム W-8BEN 入力
  ✓ 国籍：日本
  ✓ 住所：日本の住所を入力
  ✓ US TIN（Tax ID）：空白 → 「該当しません」
  ✓ その他は Amazon が指示に従う
  
↓

「送信」をクリック

↓

✅ 完了（数日～数週間で確認メール）
```

## 🔐 セキュリティのベストプラクティス

### **パスワード管理**

```
❌ 避けるべき：
  - ブラウザに「パスワードを保存」
  - 複数サイトで同じパスワード
  - 紙にメモ
  
✅ 推奨：
  - パスワード管理アプリを使用
  - 複雑なパスワード（16 文字以上推奨）
  - 定期的に変更（3 ヶ月ごと）

💡 推奨アプリ：
  - 1Password（有料，高信頼）
  - LastPass（freemium）
  - Bitwarden（完全無料）
```

### **2 要素認証（2FA）の有効化**

```
Amazon アカウント
  → Amazon セキュリティ設定
  → 「2 段階認証」を有効化

これにより：
  ✅ パスワード盗難時も安全
  ✅ メールと SMS の二重確認
  ✅ わずか 2 分で設定完了
```

### **不正アクセス対策**

```
定期的に実施：
  ✅ ログイン履歴を確認
  ✅ 知らない場所からのログインは即座に更新
  ✅ パスワードをリセット（隔月）
```

## ❓ よくあるトラブル（このステップ）

### **Q: メール認証メールが届かない**

```
A: 
1. スパムフォルダを確認
   → GMail/Yahoo メール の「迷惑メール」フォルダ
   
2. メールアドレスを間違えていないか確認
   → KDP 登録時の入力内容を確認
   
3. 再送信
   → Amazon ログインページ
   → 「メールをもう一度送信」をクリック
```

### **Q: 銀行口座情報がエラー**

```
A:
1. 入力内容を確認
   ✓ 口座名義 = 銀行登録と同じ？
   ✓ 口座番号 = 7～8 桁の数字？
   
2. 銀行に直接確認
   ✓ 支店名 = 漢字表記？
   ✓ 口座種別 = 普通？
   
3. 再入力
```

### **Q: アカウントが「限定登録」と表示される**

```
A: Amazon が追加情報を求めています。
   通常 1～2 週間で解除されます。

対応:
  1. メールを確認（Amazon から指示が届いている可能性）
  2. 追加資料を提出（本人確認書類など）
  3. Amazon サポートに連絡
     → 「アカウントについて」→「サポートに連絡」
```
